## 참고 이슈

### [정적 분석] astro:content의 z import deprecated 힌트

- **위치**: `src/content.config.ts:1`
- **설명**: Astro 콘텐츠 컬렉션 스키마 정의에서 `z`를 `astro:content`에서 직접 import하는 패턴이 deprecated 힌트(TS6385)를 발생시킨다. 9건 모두 동일 파일에 집중되어 있으며 1.4.0 사이클 이전부터 존재하던 기존 힌트다. 기능 동작에 영향 없음.
- **개선 방향**: `zod`를 직접 import하는 방식으로 교체 가능. 단, Astro 공식 문서에서 `z`를 re-export하는 패턴이 deprecated로 확정되면 그때 일괄 수정 권장.

### [E2E 테스트] 모바일 헤더 H3 시나리오 skip

- **위치**: `tests/e2e/flows/migration-flow.spec.ts:232`
- **브라우저**: mobile-chrome, mobile-safari
- **설명**: 헤더 "마이그레이션 & 진단" 링크는 `md:flex` 이상 뷰포트에서만 노출된다. 모바일(375px)에서는 햄버거 메뉴로 접혀 테스트가 skip 처리된다. 총 5건 skip (2 브라우저 × 1 케이스 + 기존 모바일 관련 skip).
- **개선 방향**: 모바일 햄버거 메뉴 열기 → 링크 클릭 시나리오로 확장 가능. 현재 1.4.0 범위 밖이므로 후속 작업으로 남긴다.
