**총 25건 경고 — 성격별 분류:**

| 파일 | 건수 | 성격 |

| `scripts/fetch-rss.ts` | 4건 | Node 스크립트 — 진행 로그 |
| `scripts/generate-favicon-ico.ts` | 1건 | Node 스크립트 |
| `scripts/generate-og-image.ts` | 1건 | Node 스크립트 |
| `scripts/generate-summary.ts` | 6건 | Node 스크립트 |
| `scripts/update-articles.ts` | 10건 | Node 스크립트 |
| `src/lib/utils/shareUrl.ts` | 2건 | 런타임 콘솔 출력 |

- `scripts/` 는 빌드/CI 전용 Node 스크립트로, console 출력이 운영 목적에 부합한다. ESLint override(`/* eslint-disable no-console */`)를 추가하거나 flat config에서 `scripts/` 경로를 예외 처리하면 경고를 제거할 수 있다.
- `src/lib/utils/shareUrl.ts` 의 console 2건은 런타임 코드에 포함되어 있어 프로덕션 환경에서 불필요한 출력을 유발한다. 제거 또는 `import.meta.env.DEV` 가드 처리 권장.

#### non-null assertion (1건, `@typescript-eslint/no-non-null-assertion`)

- **위치**: `scripts/fetch-rss.ts:380`
- `!` 단언 사용. scripts/ 범주이므로 심각도는 낮으나, 타입 가드(`if (value != null)`) 또는 nullish coalescing으로 대체 가능.
