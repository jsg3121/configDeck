SPEC-0006-design §4.2 / §5

AdvisoryDiagnosis 진단기 상태 갤러리

진단기 컴포넌트의 5가지 상태(IDLE, AFFECTED, SAFE, ERROR, DISABLED)와 package.json 붙여넣기 탭, 인터랙티브 시연을 한 페이지에 배치합니다. 시안 검토용으로만 사용되며, 실제 코드 적용 시에는 `AdvisoryDiagnosis.svelte`로 분리됩니다.

인터랙티브 시연

실제 입력 → 상태 전환

다음 입력값으로 각 상태를 시연할 수 있습니다: 14.2.20 (AFFECTED) · 15.2.5 (SAFE) · 14.abc (ERROR) · 빈 값(IDLE 유지)

ⓘ next 패키지 버전을 입력하면 영향 여부를 즉시 확인합니다.

ⓘ next 패키지의 버전을 자동으로 추출합니다.

이 진단 결과는 정보 제공 목적이며 무보증(AS IS)으로 제공됩니다. 실제 보안 위험 평가는 npm audit, Snyk, Dependabot 등의 공식 도구와 관리자 검토를 병행하세요. 공식 advisory에서 확인 ↗

상태별 시각 시안

각 상태의 결과 카드 디자인을 한눈에 비교합니다.

상태 1 / IDLE

입력 전 (idle)

결과 영역은 비어있고, 면책조항만 하단에 노출됩니다.

이 진단 결과는 정보 제공 목적이며 무보증(AS IS)으로 제공됩니다. (idle 상태에서도 표시)

상태 2 / AFFECTED

영향받음

!

영향받음

next@14.2.20은 이 취약점의 영향을 받습니다.

영향 범위: >=14.0.0 <14.2.25

권장 패치: next@14.2.25 이상

npm install next@14.2.25
이 진단 결과는 정보 제공 목적이며 무보증(AS IS)으로 제공됩니다.

상태 3 / SAFE

안전함

✓

안전함

next@15.2.5는 이 취약점의 영향을 받지 않습니다.

패치 버전 15.2.3 이상이므로 안전합니다. 최신 버전을 계속 유지하세요.

이 진단 결과는 정보 제공 목적이며 무보증(AS IS)으로 제공됩니다.

상태 4 / ERROR

입력 오류

⚠ 올바른 버전 형식이 아닙니다. 예: 14.2.20

i

입력 오류

유효한 semver 형식으로 입력해주세요. 예: 14.2.20, 15.0.0-canary.1

이 진단 결과는 정보 제공 목적이며 무보증(AS IS)으로 제공됩니다.

상태 5 / DISABLED — archived 또는 superseded

진단기 비활성

archived 사유

이 advisory가 다루는 버전은 EOL입니다. 최신 Next.js 버전으로 마이그레이션을 권장합니다.

최신 버전 확인 →

superseded 사유

이 advisory는 후속 advisory로 대체되었습니다. 진단은 후속 advisory에서 이용하세요.

후속 advisory 진단하기 →
이 진단 결과는 정보 제공 목적이며 무보증(AS IS)으로 제공됩니다. (DISABLED 상태에서도 면책 표기 유지)
